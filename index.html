Задание: Необходимо создать сайт отчет и залить его на хостинг GitHyb Pages.

Действия по создание и деплою на хостинг. Был создан новый репозиторий на сайте github pages. Была создана новая папка, в которой хранятся рабочие файла сайта.

Команды для деплоя сайта на хостинг:

git init - создает локальный репозиторий в нашей папке

git add . - добавление всех новых файлов в отслеживание

git commit -m"commit text" - создания комментария к нашим изменениям

git remote add origin ссылка_на_репозиторий - связываем наш локальный репозиторий с удаленным

git push -u origin master - пушим наши зменения на удаленный репозиторий

git checkout -b gh-pages - создаем новую ветку gh-pages, которая является хостингом

git checkout master - возвращаемся на ветку master

git push origin master:gh-pages - пушим все файлы из ветки msater в ветку gh-pages, тем самым заливая файлы на хостин
